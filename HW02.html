Or we have have the histograms with 14 bins, so that will be like:

ggplot(data = states, mapping = aes(x = income)) + 
  geom_histogram(breaks = seq(3000, 6500, 250), 
                 col = "red", fill = "cadetblue3") +

  ggtitle("Frequency histogram with 14 bins")

For the 7 and 14 bins of histogram, I prefer the 14th, because it looks more clear. And with higher frequencies you will see different sections of income more clear.

5. kernel density plots

Base R kernel density plot

plot(density(income), 
     xlab = "US states income in 1974 (base R histogram)", 
     main = "Kernel Density plot")

ggplot2 kernel density plot

ggplot(data = states, mapping = aes(x = income)) + 
  geom_density(col = "steelblue", lwd = 1.05) + 
  xlab("US states income in 1974 (ggplot2 histogram)") + 
  ggtitle("Kernel Density plot")

The difference between kernel density plot and histogram is that the histogram looks more like being visualized by the discrete data, but kernel density plot looks more like a continuous data. The kernel density plot is more likely to separate the bins in the histogram so that every “bin” of kernel density plot is some close number, so that the kernel looks to be more smooth. Histogram looks like a Riemann sum of the kernel density plot.

6. QQ plots

Base R QQ plot

qqnorm(y = income, 
       ylim = c(3000, 6500), 
       main =  "Normal QQ plot for State Income in 1974 (base R)")
qqline(y = income)

ggplot qq plot

ggplot(data = states, mapping = aes(sample = income)) + 
  geom_qq(distribution = stats::qnorm) +
  geom_qq_line(distribution = stats::qnorm) +
  xlab("Theoretical normal quantiles") + 
  ylab("Samples quantiles") + 
  ggtitle("Normal QQ plot for state Income in 1974 (ggplot)")

The QQ plot indicates the hypothesis that the distribution of income is normally distributed. The plots showed how scattered each point is aligned with the straight line. As each point is close to the straight line, that means the income is more likely to be normal distributed. As we can see from this plot, each point is close to the straight line, except the biggest num, which would be a outlier if we used in boxplot. We can also check if the income is normally distributed by using Shapiro-wilk normality test:

shapiro.test(income)
## 
##  Shapiro-Wilk normality test
## 
## data:  income
## W = 0.9769, p-value = 0.43

We can see the value of W is close to 1, and the p-value is lower than 0.5, which tells us that the sample data is almost normally distributed.
